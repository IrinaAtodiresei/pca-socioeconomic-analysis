# Socio-Economic Data Analysis using PCA

This project applies Principal Component Analysis (PCA) to socio-economic indicators from the World Bank (2022) to reduce dimensionality and identify latent development dimensions.

## Dataset
- Source: World Bank Open Data (World Development Indicators)
- 240 countries
- 8 socio-economic indicators

## Project Workflow
- Data cleaning and preprocessing
- Long-to-wide pivot transformation
- Missing value handling (median imputation)
- Standardization (z-score scaling)
- Principal Component Analysis (PCA)
- Scree plot and 2D visualization
- Export of analytical results

## Technologies Used
- Python 3
- pandas
- scikit-learn
- matplotlib

## Key Results
- First principal component explains ~58% of total variance
- First two components explain ~71% of total variance
- PCA captures a general socio-economic development dimension
